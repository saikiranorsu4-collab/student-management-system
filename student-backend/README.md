# Student Management REST API

A simple RESTful API built using Node.js, Express.js, and MongoDB for managing student data.

## Features

- Create Student
- Get All Students
- Update Student
- Delete Student

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Thunder Client

## API Endpoints

### Create Student
POST /api/students

### Get All Students
GET /api/students

### Update Student
PUT /api/students/:id

### Delete Student
DELETE /api/students/:id

## Installation

```bash
npm install
npm run dev
